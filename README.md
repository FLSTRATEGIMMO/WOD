# 💪 WOD BOOK — Niveau Confirmé

Application web de suivi d'entraînement militaire fonctionnel.  
Accessible sur PC et smartphone, synchronisation automatique entre appareils.

## 🔗 Accès en ligne
> **[Ouvrir l'application →](https://[VOTRE-USERNAME].github.io/wod-book)**

---

## 📋 Contenu
- **150 WODs** : 50 Haut du corps · 50 Bas du corps · 50 Corps complet
- **50 Challenges** : tests de sélection, records à battre
- Niveau : **Confirmé → Élite**

## ⚙️ Fonctionnalités
- Filtres par catégorie, recherche par exercice
- Enregistrement date / temps / score / ressenti / note
- Logbook complet avec historique
- Barres de progression par catégorie
- Synchronisation PC ↔ Smartphone (même prénom + code équipe)
- 100% offline après premier chargement

## 🚀 Déploiement
Hébergé via **GitHub Pages** — aucun serveur requis.
